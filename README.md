# 🎬 Movie Recommendation System

A movie recommendation system built using Python, Pandas, Scikit-Learn, and Streamlit.

## Features

* Select a movie from the dropdown
* Get 5 similar movie recommendations
* Interactive Streamlit web app
* Uses the MovieLens dataset

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Streamlit
* Pickle

## Dataset

MovieLens Small Dataset

* 9,742 Movies
* 100,836 Ratings

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/movie-recommendation-system.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the app

```bash
streamlit run app.py
```

## Project Workflow

* Loaded movie and ratings datasets
* Merged and cleaned data
* Created a movie-user pivot table
* Calculated cosine similarity between movies
* Built a recommendation engine
* Developed a Streamlit interface
